Series¶

A simple definition of Series: it's a 1-dimensional array.

To create your own Series, you can input pd.Series() with arguments data=[list_of_values], index=[list_of_index]

Take note that the index indicates the position of the row.

In the next cell, you will see an example of a Series with sample grades as values

In [2]:
s_grades = pd.Series(data=[90,85,93,77,88], index=[0,1,2,3,4])
s_grades
Out[2]:
0    90
1    85
2    93
3    77
4    88
dtype: int64

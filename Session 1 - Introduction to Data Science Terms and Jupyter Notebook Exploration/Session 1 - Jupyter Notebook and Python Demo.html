Initialization vs Declaration¶

Initialization = process of assigning a value to a variable
Declaration = process when you only declare a datatype in a variable.

Remember: Python doesn't need a datatype to determine the variable's type. It automatically assigns its data type based on the assigned value to it.

In [4]:
x=10 # this is an initialization 
y=0 # this is also an initialization

Just think of a variable as a storage.
The value of our variable x above indicates that it contains a value of 10

Run the cell below to display the value of x:

In [5]:
print(x)

10

= vs ==¶

= means assignment operator
An assignment operator is used to assign a value from right operand to your left operand.
Example: x = 10
Explanation: This means that we assign a value of 10 to our variable x

== means comparison operator A comparison operator is used to compare values from left and right.
Example: 100==200
Explanation: This means that the value of 100 and 200 are being compared to each other. This will return either True or False once the expression is run at a cell.
